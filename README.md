##  Installation
```bash
npm i
```
![📹 Installation using command line.](./assets/install.gif)

### this will install the following dependencies 
```bash
  "dependencies": {
    "dotenv": "^10.0.0",
    "express": "^4.17.1",
    "mysql2": "^2.3.0",
    "nodemon": "^2.0.12",
    "sequelize": "^5.22.4"
  }

```

## 📚
## Run the app

  To use this app, you may need to run the following commands sequencially,

```bash
* mysql -u root -p
* source db/schema.sql;
* quit

* npm run seed

* npm run dev
or
* npm run start
```
